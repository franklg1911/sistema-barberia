# Sistema de Gestión para Barberías

Este es un sistema de gestión diseñado para barberías que permite registrar ventas, gastos y generar reportes mensuales de ganancias. Desarrollado con PHP, MySQL, HTML, CSS, JavaScript y Bootstrap.

---

## Características principales
- **Registro de ventas**: Permite registrar cada venta con detalles como servicio, precio y cliente.
- **Registro de gastos**: Facilita el registro de gastos diarios.
- **Reportes mensuales**: Genera reportes de ganancias netas, ventas y gastos.
- **Autenticación segura**: Sistema de login para proteger el acceso al sistema.

---

## Tecnologías utilizadas
- **Frontend**: HTML, CSS, JavaScript, Bootstrap.
- **Backend**: PHP.
- **Base de datos**: MySQL.
- **Control de versiones**: Git.
